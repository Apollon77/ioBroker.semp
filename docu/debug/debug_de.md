# Debug-Möglichkeiten


## ioBroker log

Die einfachste Möglichkeit, Fehler und Informationen über dern Adapter zu finden, ist das debug log.
Dazu zunächts die Protokollstufe des Adapters auf debug stellen, den Adapter neu starten.
Unter Protokolle findet man dann die logs.
